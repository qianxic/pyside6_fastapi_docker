- **软件名称**：遥感影像变化检测系统
- **英文名称**：Remote Sensing Image Change Detection System
- **版本号**：V1.0

## 技术特点

- **前端技术**：PySide6 + Qt框架 + 响应式布局
- **后端技术**：Python + FastAPI + PyTorch + CUDA
- **深度学习**：PyTorch 2.8.0 + X3D模型
- **地理信息**：GDAL + GeoPandas + Rasterio
- **容器化**：Docker + Docker Compose + NVIDIA Docker

## 核心功能

1. **变化检测引擎**
   - 单图像变化检测
   - 栅格影像变化检测
   - 批量图像处理
   - 批量栅格处理

2. **图像处理模块**
   - 多格式支持（PNG、JPG、TIFF、GeoTIFF等）
   - 图像交互（缩放、平移、区域选择）
   - 地理信息保持
   - 数据预处理

3. **用户界面模块**
   - 四区域布局设计
   - 深色/浅色主题切换
   - 实时日志显示
   - 进度可视化

4. **批量处理模块**
   - 任务队列管理
   - 并发处理支持
   - 结果导出（Shapefile、GeoJSON）
   - 内存优化

## 创新点

1. **X3D深度学习架构**
   - 基于3D卷积神经网络的变化检测算法
   - GPU加速的深度学习推理引擎
   - 自适应模型参数优化机制

2. **地理信息保持技术**
   - 栅格影像地理坐标系统维护
   - 矢量数据空间参考一致性保证
   - 多尺度地理信息融合算法

3. **异步并发处理架构**
   - 基于FastAPI的高性能异步服务
   - 多任务并行处理机制
   - 智能负载均衡与资源调度
